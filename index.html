    如果您已经完成以下准备工作，可以直接从 配置 Hexo 博客 开始。

    2. 去 Node.js 官网 下载安装 Node.js，其中最后一个空建议选 npm，方便后续操作。

    3. 注册 github 账号：

      1. 打开 github 注册页。如果打不开或者加载很慢的话，可以搜索 steam++；实在不行就自行百度其他方法吧。

      2. 如图填写相应内容进行注册。其中邮箱的话 QQ 邮箱就行（对于没有邮箱的人群），至于如何注册邮箱，请自行百度。最好（或是说一定）不要用临时邮箱！
