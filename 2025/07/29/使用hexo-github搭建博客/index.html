    这篇文章更倾向于是一篇博客搭建经历，可能并不能作为一个很好的教程。
    全文涉及网站大多均为英语，不善英语者可安装翻译器浏览器插件

    我很早之前就有过一个想要搭建自己博客的想法，但因为电脑是32位一直没能实现。
    直到这天，我突然想起我的树莓派，它事 64 位的，也就是说它可以完成这些操作，于是我便开始了实践……

    • 准备工作

    如果您已经完成以下准备工作，可以直接从配置（初始化）Hexo博客开始。

    1. 去git官网下载安装git。
    2. 去Node.js官网下载安装Node.js，其中最后一个空建议选npm，方便后续操作。
    3. 注册github账号：
      1. 打开github注册页。如果打不开或者加载很慢的话，可以搜索steam++；实在不行就自行百度其他方法吧。
      2. 如图填写相应内容进行注册。其中邮箱的话QQ邮箱就行（对于没有邮箱的人群），至于如何注册邮箱，请自行百度。最好（或是说一定）不要用临时邮箱！
    4. 打开命令行，输入npm install -g hexo-cli安装Hexo。
